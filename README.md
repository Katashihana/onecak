# onecak
An unofficial API of 1cak.com

## Usage
Lol -> 10 Recent stored posts on database  
Example: ```https://onecak.herokuapp.com/?lol```

Shuffle -> Select random posts from database  
Example: ```https://onecak.herokuapp.com/?shuffle=5```
